# Consequences-of-Sleep-Duration
This project analyzes the impact of sleep duration on various health and lifestyle factors using a publicly available sleep health dataset from Kaggle. The analysis explores relationships between daily sleep hours and key variables such as sleep quality, stress level, physical activity, BMI category, and sleep disorders. Through exploratory data analysis and visualizations, the study highlights patterns showing how insufficient or excessive sleep may be associated with reduced sleep quality and increased stress, emphasizing the importance of balanced sleep for overall well-being.
